LeoVue
======

A Vue.js components lib by Leo.

- [LeoVue](https://axolo.github.io/leovue)


Files
-----
```
|-- dist          # npm publish
|-- docs          # docs dist (Pages Service)
|-- manual        # docs src
|-- labs          # labs src
`-- src           # leovue src
```

Install
-------
```bash
npm install @axolo/leovue --save
```

Use
---
```js
import { LeoDialog, LeoUpload } from '@axolo/leovue'
```
